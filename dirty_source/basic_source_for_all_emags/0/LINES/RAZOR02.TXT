                          -*- Installation Notes -*-

       Q. What was ripped and why?
       A. Well, as usual CD-Audio and Movies had to go...peace out!

       Simple as always, simply unzip the files you downloaded to a temp dir
       then create a directory you want the game to be held in.  After that,
       simply unRAR (or unWINRAR) to the directory you previously created.

       (eg:) UNRAR.EXE x -y -v bugrider.001 c:\games\bugrider

       After un-RARing the game, run SETUP.EXE to setup your registry
       settings. You can then run SPLASH.EXE to start the game. From
       the Splash screen you can click the "Configure" button to change
       settings in the game relating to keyboard/joystick control and
       most importantly, 3Dfx support. =) We've also included the
       DirectPlay v5.0a update and a newer version of MSVCRT.DLL in
       the EXTRAS directory. If you haven't installed these updates,
       it is recommended that you do so. Enjoy!
